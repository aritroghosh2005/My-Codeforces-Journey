The game consists of $$$3$$$ rounds, in each round both players score some points (from $$$0$$$ to $$$k$$$), and for each round, Alice's score differs from Bob's score. The player who scores more points in a round is considered the winner of that round.</p><p>In the first round, Alice scored $$$a_1$$$ points, and Bob scored $$$b_1$$$. In the second round, Alice scored $$$a_2$$$ points, and Bob scored $$$b_2$$$.</p><p>The winner of the game is the one who has the higher total score. If Alice's total score equals Bob's total score, the player who won more rounds is declared the winner. Alice wants to understand if Bob has a chance to win, or if she will definitely win regardless of the results of the $$$3$$$-rd round. Help her determine this!</p><p>Please note that in each round, a player can score at least $$$0$$$ and at most $$$k$$$ points. Additionally, for each round, Alice's score differs from Bob's score.</p></div><div class="input-specification"><div class="section-title">Input</div><p>The first line contains a single integer Alice and Bob respectively in the first round; </li><li> the third line contains two integers $$$a_2$$$ and $$$b_2$$$ ($$$0 \le a_2, b_2 \le k$$$; $$$a_2 \ne b_2$$$) — the points scored by Alice and Bob respectively in the second round. </li></ul></div><div class="output-specification"><div class="section-title">Output</div><p>For each test case, output <span class="tex-font-style-tt">NO</span>
