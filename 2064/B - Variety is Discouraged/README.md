<div class="header"><div class="title">B. Variety is Discouraged</div><div class="time-limit"><div class="property-title">time limit per test</div>1.5 seconds</div><div class="memory-limit"><div class="property-title">memory limit per test</div>256 megabytes</div><div class="input-file input-standard"><div class="property-title">input</div>standard input</div><div class="output-file output-standard"><div class="property-title">output</div>standard output</div></div><div><p>Define the score of an arbitrary array $$$b$$$ to be the length of $$$b$$$ minus the number of distinct elements in $$$b$$$. For example: </p><ul> <li> The score of $$$[1, 2, 2, 4]$$$ is $$$1$$$, as it has length $$$4$$$ and only $$$3$$$ distinct elements ($$$1$$$, $$$2$$$, $$$4$$$). </li><li> The score of $$$[1, 1, 1]$$$ is $$$2$$$, as it has length $$$3$$$ and only $$$1$$$ distinct element ($$$1$$$). </li><li> The empty array has a score of $$$0$$$. </li></ul><p>You have an array $$$a$$$.
